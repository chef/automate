API

Specify whether the checkbox is initialized checked. Specify whether the checkbox is initialized disabled. Catch the checkValue event emitted when the checkbox changes. Specify text that comprises the rendered label of the <chef-checkbox>.

Accessibility

The chef-checkbox wraps the mat-checkbox component. mat-checkbox uses an internal input type="checkbox" to provide an accessible experience. This internal checkbox receives focus and is automatically labelled by the text content of the mat-checkbox element.

Checkboxes without text or labels should be given a meaningful label via aria-label or aria-labelledby.

See Angular Material docs.

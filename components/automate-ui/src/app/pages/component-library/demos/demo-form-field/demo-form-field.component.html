chef-form-field is a component used to wrap several chef form components to apply some common styles and behaviors. chef-form-field is heavily inspired by Angular material's mat-form-field. The chef-form-field is designed to wrap <input matInput> & <textarea matInput> components.

Angular Template Code Example

<chef-form-field>
  <label>Test Field<label>
  <input chefInput #testfield id="test" required type="text" placeholder="Type 'error' to see an example error.">
  <chef-error *ngIf="testfield.value == 'error'">This is an example error</chef-error>
</chef-form-field>

API

Flow content; typically <input> and <textarea> elements.

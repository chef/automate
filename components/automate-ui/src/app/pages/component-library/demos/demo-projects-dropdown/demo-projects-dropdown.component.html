API

The map of ProjectChecked by id. Any checked changes propagated via onProjectChecked. Updates should be applied to parent component state. Emits an updated project that has been checked or unchecked. Keep projects input up to date in parent. Specify whether the dropdown should be disabled.

Accessibility

This component is keyboard accessible. You can use the tab key to navigate to the component and press enter/spacebar to open the dropdown. Then you can use the tab key (or shift+tab to move backwards) to navigate to the checkboxes in the dropdown. Use the space bar to check/uncheck the checkboxes. Use the enter key to close the dropdown.

You can also navigate between the checkboxes using the up/down arrow keys. You cannot use the arrow keys to navigate between the button (the element that says 'select projects') and the checkboxes. The whole component will be navigable with arrow keys in future iterations.
